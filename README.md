# 🔥 Uchiha Itachi – Sharingan Hand Seals

An interactive anime-inspired hand-tracking visualization powered by **Three.js** and **MediaPipe Hands**. Perform different hand signs to activate iconic Uchiha visual effects including Fire Style Jutsu, Tsukuyomi, Genjutsu Seals, and Mangekyō Sharingan.

## ✨ Features

* 🖐️ Real-time hand gesture recognition
* 👁️ Mangekyō Sharingan visual effects
* 🔥 Fire Style: Fire Ball Jutsu particle system
* 🌙 Tsukuyomi illusion visualization
* 🌀 Anime-inspired Genjutsu magic seals
* ☁️ Dynamic particle morphing engine
* 💫 Bloom and cinematic post-processing
* 🎥 Live webcam hand tracking
* 🎮 Manual Jutsu Control Panel
* 🌌 25,000+ animated particles

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Three.js
* MediaPipe Hands
* MediaPipe Camera Utils
* Unreal Bloom Pass
* WebGL

## 🎮 Gesture Controls

| Gesture                  | Technique          |
| ------------------------ | ------------------ |
| No Hand Detected         | Chakra Neutral     |
| Two Hands Raised         | Fire Ball Jutsu    |
| Index + Middle Finger Up | Tsukuyomi          |
| Index Finger Only        | Genjutsu Seal      |
| Thumb + Index Pinch      | Mangekyō Sharingan |

## ⚔️ Available Jutsu

### 🔥 Fire Style: Fire Ball Jutsu

Creates a massive rotating vortex of fiery particles.

### 🌙 Tsukuyomi

Projects hypnotic crimson rings and illusion effects.

### 🌀 Genjutsu Anime Seal

Generates magical circles, pentagrams, and glowing illusion symbols.

### 👁️ Mangekyō Sharingan

Activates Itachi's iconic Sharingan-inspired rotating visual pattern.

### ☁️ Chakra Neutral

A calm idle state with ambient crimson particle effects.
